# Alternative Assets for Star Office UI

Generated: 2026-03-20
Model: nano-banana-pro-preview (Gemini)

These are alternative decoration assets for the Star Office "Decorate Room" feature.
They do NOT replace any current assets — they're options to swap in.

## Backgrounds (16:9, room decoration)

All backgrounds are WebP format, suitable for the `office_bg` slot.

- `alt-bg-tavern.webp` — Cozy medieval tavern with stone walls, wooden beams, fireplace glow
- `alt-bg-treehouse.webp` — Enchanted forest treehouse workshop with glowing mushrooms
- `alt-bg-captain-cabin.webp` — Ship captain's cabin with maps, lantern light, portholes
- `alt-bg-forge.webp` — Underground forge workshop with glowing embers, anvil, stone walls

## Posters (160x160 PNG, for poster spritesheet slots)

Each is a standalone 160x160-ish pixel art piece. To use in the poster spritesheet,
they'd need to be composited into the spritesheet grid.

- `alt-poster-crab-fire.png` — Pixel art crab with fire/flame motif (Cali!)
- `alt-poster-hearth-welcome.png` — "Welcome to The Hearth" wooden sign
- `alt-poster-minnesota.png` — Minnesota landscape with lakes, pines, northern lights
- `alt-poster-coffee.png` — Steaming coffee cup, cozy warm tones
- `alt-poster-glowing-books.png` — Bookshelf with glowing magical books
- `alt-poster-crab-crest.png` — Family crest with crabs and fire motifs

## Memo Backgrounds (for memo/note panel)

WebP format, suitable for the `memo_bg` slot.

- `alt-memo-leather.webp` — Warm leather-bound journal page
- `alt-memo-chalkboard.webp` — Dark green chalkboard with wooden frame
- `alt-memo-corkboard.webp` — Cork bulletin board with pushpins

## Usage Notes

- Backgrounds can be swapped via the Decorate Room UI or the asset drawer
- Posters need to be resized to exactly 160x160 and composited into a spritesheet for use
- Memo backgrounds can replace `memo-bg.webp` directly
- All generated with pixel art style prompts for consistency with the Star Office aesthetic
